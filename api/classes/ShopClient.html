Proxy to serializer's deserialize.

  • payload Object

    The raw payload returned by the adapter.

  • adapter BaseAdapter

    The adapter that yielded the payload.

  • existingSerializer BaseSerializer

    The serializer to attach. If none is passed, then this.deserialize will create one for the type.

  • opts Object

    Options that determine which deserialization method to use.

    • multiple Boolean

      true when the payload represents multiple models

    • single Boolean

      true when the payload represents one model.
