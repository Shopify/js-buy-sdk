ShopClient public static

Defined in src/shopify.js:20

Create a ShopClient. This is the main entry point to the SDK.

const client = ShopifyBuy.buildClient({
  apiKey: 'abc123',
  channelId: 123456,
  myShopifyDomain: 'myshop'
});

Parameters:

  • configAttrs Object

    A hash of required config data.

    • apiKey String

      Your api client's public token.

    • channelId String

      The channel from which to read publications. Visit <your-shops-domain>/admin/channels.json while authenticated to see a list of available channels.

    • myShopifyDomain String

      You shop's myshopify.com domain.

Returns:

ShopClient:

a client for the shop using your api credentials.
