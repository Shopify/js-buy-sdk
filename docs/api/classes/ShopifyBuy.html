Defined in src/shopify.js:20

Create a ShopClient. This is the main entry point to the SDK.

const client = ShopifyBuy.buildClient({
  apiKey: 'abc123',
  appId: 123456,
  myShopifyDomain: 'myshop'
});

Parameters:

  • configAttrs Object

    An object of required config data.

    • apiKey String

      Your api client's public token.

    • appId String

      The app whose listings the client will be using. If you are just modifying a buy button, the buy-button's app id is

      1. Otherwise, obtain the app id of the app you're modifying or extending.
    • myShopifyDomain String

      You shop's myshopify.com domain.

Returns:

ShopClient:

a client for the shop using your api credentials.
